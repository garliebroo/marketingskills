# SEO Basics

A foundational skill for understanding and applying Search Engine Optimization principles to improve organic visibility.

## Description

Search Engine Optimization (SEO) is the practice of optimizing web content and structure to rank higher in search engine results pages (SERPs). This skill covers the core concepts every marketer should know.

## Key Concepts

### On-Page SEO
- **Title Tags**: Keep under 60 characters, include primary keyword near the front
- **Meta Descriptions**: 150-160 characters, compelling summary that drives clicks
- **Header Hierarchy**: Use H1 for page title, H2/H3 for sections — one H1 per page
- **Keyword Placement**: Natural usage in first 100 words, headers, and throughout body
- **Image Alt Text**: Descriptive text for accessibility and image search indexing

### Technical SEO
- **Page Speed**: Aim for Core Web Vitals scores in the "Good" range
- **Mobile-First**: Google indexes mobile version first — always test on mobile
- **Crawlability**: Ensure robots.txt isn't blocking important pages
- **Sitemap**: Submit XML sitemap to Google Search Console
- **HTTPS**: Secure sites get a minor ranking boost and user trust

### Off-Page SEO
- **Backlinks**: Quality over quantity — links from authoritative, relevant sites matter most
- **Anchor Text**: Varied, natural anchor text pointing to your pages
- **Brand Mentions**: Even unlinked mentions contribute to brand authority signals

## Common Mistakes to Avoid

1. **Keyword stuffing** — Google penalizes unnatural keyword repetition
2. **Duplicate content** — Use canonical tags when similar content must exist
3. **Ignoring search intent** — Match content format to what users actually want
4. **Buying low-quality backlinks** — Can result in manual penalties
5. **Skipping internal linking** — Helps distribute page authority across your site
